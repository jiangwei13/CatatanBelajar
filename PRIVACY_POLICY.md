# PRIVACY POLICY - CATATAN BELAJAR

**Terakhir Diperbarui:** 04 Januari 2026

## 1. PENDAHULUAN

Pulo Torang ("perusahaan kami") menghormati privasi pengguna aplikasi "Catatan Belajar" ("Aplikasi"). Privacy Policy ini menjelaskan bagaimana Aplikasi menangani informasi pengguna.

## 2. INFORMASI YANG TIDAK DIKUMPULKAN

Aplikasi Catatan Belajar adalah aplikasi **100% offline** yang **TIDAK mengumpulkan, menyimpan, atau mengirimkan data pribadi pengguna** ke server manapun. Aplikasi ini:

- **TIDAK** mengumpulkan informasi pribadi (nama, email, nomor telepon, dll)
- **TIDAK** menggunakan layanan analytics atau tracking
- **TIDAK** menampilkan iklan
- **TIDAK** memerlukan koneksi internet
- **TIDAK** mengirimkan data ke pihak ketiga
- **TIDAK** menggunakan cookies atau teknologi pelacakan lainnya
- **TIDAK** memerlukan registrasi atau login

## 3. DATA YANG DISIMPAN SECARA LOKAL

Aplikasi menyimpan data catatan belajar pengguna **hanya di perangkat pengguna** menggunakan penyimpanan internal Android (SharedPreferences dan Internal Storage). Data yang disimpan meliputi:

- Judul catatan
- Tema belajar hari ini
- Tanggal dan waktu
- Durasi/waktu belajar
- Isi catatan
- Status pemahaman (Sudah Paham, Belum Paham, Butuh Review)
- Kategori catatan
- Lampiran gambar (jika ada)

**Semua data ini disimpan secara lokal di perangkat pengguna dan tidak pernah dikirim ke server atau pihak ketiga manapun.**

## 4. PERIZINAN APLIKASI

Aplikasi ini **tidak memerlukan izin khusus** dari pengguna. Aplikasi hanya menggunakan:

- **Akses penyimpanan internal** (untuk menyimpan catatan dan gambar lampiran) - ini adalah penyimpanan privat aplikasi yang tidak memerlukan izin khusus
- **Pemilih gambar** (untuk menambahkan lampiran) - menggunakan sistem pemilih gambar Android yang tidak memerlukan izin penyimpanan

Aplikasi **TIDAK** meminta izin untuk:
- Lokasi
- Kamera
- Mikrofon
- Kontak
- Kalender
- Telepon
- SMS
- Internet

## 5. BACKUP DATA ANDROID

Aplikasi mengaktifkan fitur backup Android default. Data aplikasi dapat di-backup oleh sistem Android ke Google Drive pengguna (jika pengguna mengaktifkan backup Android). Backup ini dilakukan oleh sistem Android, bukan oleh aplikasi kami. Data yang di-backup tetap berada di akun Google pengguna dan tidak diakses oleh kami.

## 6. KEAMANAN DATA

Karena semua data disimpan secara lokal di perangkat pengguna:

- Data tidak dapat diakses oleh pihak ketiga
- Data tidak dapat dicuri melalui serangan jaringan
- Data akan terhapus jika pengguna menghapus aplikasi
- Pengguna memiliki kontrol penuh atas data mereka

## 7. HAK PENGGUNA

Pengguna memiliki hak penuh atas data mereka:

- **Akses Data:** Pengguna dapat melihat semua data mereka melalui aplikasi
- **Hapus Data:** Pengguna dapat menghapus catatan individual atau semua data melalui menu "Reset Semua Data" di Pengaturan
- **Ekspor Data:** Pengguna dapat membagikan catatan melalui fitur "Bagikan" di aplikasi
- **Kontrol Penuh:** Karena data disimpan lokal, pengguna memiliki kontrol penuh atas data mereka

## 8. PERUBAHAN PRIVACY POLICY

Kami berhak mengubah Privacy Policy ini kapan saja. Perubahan akan diberitahukan melalui pembaruan aplikasi. Penggunaan aplikasi setelah perubahan berarti pengguna menyetujui Privacy Policy yang baru.

## 9. PENGIRIMAN DATA KE PIHAK KETIGA

Aplikasi **TIDAK mengirimkan data apapun ke pihak ketiga**. Aplikasi berfungsi sepenuhnya offline dan tidak terhubung ke server manapun.
